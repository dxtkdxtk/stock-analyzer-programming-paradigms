To execute our program.
*Run make inside the SACudaLibrary file and pull the .so file out to the main folder.
*Execute program with: python stockGUI.py

Note:
To do the inverse trends or market average you must be on a machine with CUDA.
To pull live data you must be on a machine with a connection to the internet.
